# Semana 13 - Pruebas UI automatizadas para validaciones de socios

Estas pruebas aplican Continuous Testing al módulo de socios del sistema Gym System.

Casos automatizados:

1. Login correcto.
2. Login incorrecto.
3. Registro de socio válido.
4. DNI obligatorio.
5. DNI con 8 números.
6. Teléfono con 9 números.
7. Correo inválido.
8. Correo opcional.

El pipeline de GitHub Actions debe ejecutar estas pruebas automáticamente en cada push o pull request.
